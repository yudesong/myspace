安装说明：

 1. 打开 config 目录里的 config_MySQL.php，填入MySQL连接信息

 2. 打开 config 目录里的 config_PHPSay.php，更改相应信息！
   
   2.1 ppsecure 的值一定要改！随便填12个字符。

   2.2 QQ互联的appid和appkey需要去 http://connect.qq.com/ 申请！

 3. 设置 _cache | avatar | pictures 三个目录为0777可写！

 4. 运行 http://your.domain/install.php，安装成功后记得删除 install.php 和 update.php

 5. 默认管理员 PHPSay, 默认密码 123456
